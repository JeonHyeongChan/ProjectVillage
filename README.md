# ProjectVillage
경일아카데미 메월드 프로젝트  

Summary(required) 작성 규칙  
: []안에 커밋 메시지, 내용 한 줄 요약  
- ex) [new] 스킬 테이블 추가  

Description 작성 규칙  
: 본문 + 푸터 작성  
본문  
- What, Why를 중심으로 작성.
- 간결하게, 한 문장마다 줄 바꿈 하여 작성.

푸터  
- 변경한 내용을 확인할 수 있도록 작성.
- 변경 내용, 변경 위치.

ex)  
게임 내 스킬 적용을 위한 테이블 추가. (본문)  
- 스킬 테이블 추가 및 스테이터스 옵션 변경 (푸터)
- skilltable, a:4 오기입 수정

커밋 메시지  
- new : 새로운 파일 추가
- feat : 새로운 기능 추가
- fix : 버그 수정
- docs : 문서 변경
- style : 코드 포맷팅 (세미콜론 누락 등)
- refactor : 코드 리팩토링(기능 변경 없는 수정)
- test : 테스트 추가 또는 수정
